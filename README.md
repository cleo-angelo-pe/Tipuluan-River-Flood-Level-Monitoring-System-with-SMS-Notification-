# 🌊 Tipuluan River Flood Level Monitoring System with SMS Notification

## 📌 Overview

The **Tipuluan River Flood Level Monitoring System with SMS Notification** is a capstone project designed to provide real-time flood monitoring and early warning alerts for communities living near the Tipuluan River.

This system integrates:
- Hardware sensors  
- A web-based dashboard  
- SMS notification services  

It ensures that residents receive timely alerts when water levels rise to critical points.

---

## 🚨 Problem Statement

Communities near the Tipuluan River face several challenges:

- ❌ Lack of timely flood warnings  
- ❌ Absence of real-time water level monitoring  
- ❌ Increased risk of fatalities and property damage  
- ❌ Uncertainty and anxiety during rainy seasons  
- ❌ No centralized system for flood data tracking and alerts  

---

- 📈 Historical data tracking

- Phone number management

---

## 🔑 Key Functions

- 📌 Users must register their phone numbers to receive SMS alerts  
- 📊 View real-time water level status  
- 📁 Access historical flood data  
- ⚙️ Manage alert preferences *(future enhancement)*  

> ⚠️ Without registration in the dashboard, users will **not receive SMS notifications**.

---

## 🛠️ Technologies Used

### 🔧 Hardware
- Water level sensors  
- ESP32 microcontroller  

### 💻 Software
- SMS Gateway API  
- Web Technologies (HTML, CSS, JavaScript, PHP)  

---

## 🚀 How It Works

1. Sensors measure the river's water level  
2. ESP32 processes the collected data  
3. Data is sent to the web dashboard  
4. The system checks predefined threshold levels  
5. If critical levels are reached:
   - 📩 SMS alerts are sent to registered users  
   - 📊 Data is stored for monitoring and analysis
